HTML (Hypertext Markup Language) is a markup language used for creating web pages and applications. HTML is based on a hierarchical structure of tags that describe the content and layout of a webpage.

CSS

CSS, or Cascading Style Sheets, is a programming language used for describing the presentation of a document written in HTML or XML. It allows developers to separate the presentation layer from the content layer, providing greater flexibility and control over the appearance of a web page or application.

JavaScript

JavaScript is a high-level programming language used to create interactive and dynamic websites. It can be executed on both the client-side and server-side, making it a versatile and popular choice for web development.

NodeJS

Node.js is a JavaScript runtime built on the Chrome V8 engine that allows developers to run JavaScript code on the server-side. It provides an event-driven, non-blocking I/O model that makes it efficient and scalable for building network applications.

C++

C++ is a powerful, general-purpose programming language that is widely used in developing operating systems, game engines, and other performance-critical applications. It offers a mix of high-level abstractions and low-level memory manipulation capabilities.

Pyton

Python is a popular high-level programming language known for its simplicity, readability, and ease of use. It has a broad range of applications, from web development and data analysis to artificial intelligence and machine learning.
